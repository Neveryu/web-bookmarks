# 数组或者对象的浅拷贝&深拷贝 <Badge text="todo"/>

::: danger
数组、对象、Function 属于引用类型；基本类型不存在拷贝的问题，所以不讨论。
:::

## 数组

1、slice()

2、concat()

3、Array.from()

4、结构、赋值
``` js
var arr1 = [1,2,3]
var arr2 = [...arr1]
```

## 对象
1、Object.assign()

2、let objClone = { ...obj }

3、JSON.parse(JSON.stringify(obj))
